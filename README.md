# Dr. Neha Agarwal | ENT Specialist in Guwahati

A premium, responsive website for Dr. Neha Agarwal, a highly experienced ENT specialist and Head-Neck Surgeon based in Christian Basti, Guwahati.

## Features
- **Modern & Premium Design**: Clean, medical-focused aesthetic with interactive elements.
- **Responsive Layout**: Optimized for mobile, tablet, and desktop views.
- **Appointment Booking**: Integrated consultation scheduling form.
- **Service Listings**: Detailed information about Ear, Nose, Throat, and Pediatric care.
- **Patient Reviews**: Dynamic section showcasing patient testimonials.
- **Clinic Information**: Real-time Google Maps integration and clinic hours.
